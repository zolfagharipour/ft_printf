CONVERSIONS:


%c  -->		character
%s  -->		string
%p  -->		address
%d  -->		integer
%i  -->		integer and hex.
%u  -->		determine the correct base (decimal, octal, hexadecimal)
%x  -->		hexadecimal integer in lowercase.
%X  -->		hexadecimal integer in uppercase.


FLAGS:


%-  -->		left justifying (see *)
%0  -->		padd the number with zeroes (%03 for 1 --> 001)
%.* -->		precision of convertion (%.2f for 3.141592 --> 3.14)

*(int number can be thought as a flag which specifies the width of the string printed. if less it prints space instead and with - you can left justify)*






BONUS CONVERSIONS:


%n  -->		doesnt print anything instead store the value of number of characters printed before it.
%f  -->		floating point in decimal format (6 decimal digits)
%g  -->		floating point in either decimal (%f) or exponential (%e) format based on order of magnitude.
%e  -->		floating point in exponential format (3.140000e+00)


BONUS FLAGS:


%#	 -->	add a prefix to octal (0) and hex (0x/0X)
%(space) -->	add space before positive numbers
%+	 -->	add + before positive numbers

%l  -->		long
%ll -->		long long
%h  -->		short
%hh -->		unsigend short

**(long and short sizes are to be determined)**
